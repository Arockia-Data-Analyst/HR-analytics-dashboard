# HR-analytics-dashboard
HR Analytics Dashboard using Power BI for employee attrition and workforce analysis
This project focuses on analyzing employee attrition, workforce demographics, salary distribution, and departmental trends using Power BI.

The dashboard provides HR-related insights that help organizations understand employee retention patterns and workforce characteristics.
# TOOLS USED
• Power BI
• Data Visualization
• HR Analytics
• Business Intelligence
# DASHBOARD FEATURES
• Employee Attrition Analysis
• Department-wise Attrition
• Gender Distribution
• Salary Analysis
• Age Distribution
• Interactive Filters
• HR KPI Tracking
# KEY INSIGHTS
• Research & Development department showed the highest employee attrition.

• Most employees were concentrated between ages 30–40.

• Managerial roles received the highest average salaries.

• Male employees represented a larger workforce percentage.
